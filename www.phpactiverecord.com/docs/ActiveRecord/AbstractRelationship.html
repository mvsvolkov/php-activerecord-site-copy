Eagerly loads relationships for $models.



	
    
				
  • 
			Table
			$table -
			
		
    • 
				
  • 
			$models
			$models -
			array of model objects
		
    • 
				
  • 
			$attributes
			$attributes -
			array of attributes from $models
		
    • 
				
  • 
			$includes
			$includes -
			array of eager load directives
		
    • 
				
  • 
			$query_keys
			$query_keys -
			-> key(s) to be queried for on included/related table
		
    • 
				
  • 
			$model_values_keys
			$model_values_keys -
			-> key(s)/value(s) to be used in query from model which is including
		
    • 
			






This method takes an array of models, collects PK or FK (whichever is needed for relationship), then queries  the related table by PK/FK and attaches the array of returned relationships to the appropriately named relationship on  $models.
