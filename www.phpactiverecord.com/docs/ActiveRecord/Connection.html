 protocol://username:[email protected][:port]/dbname
 protocol://urlencoded%20username:[email protected][:port]/dbname?decode=true
 protocol://username:[email protected](/some/file/path)/dbname

	


Sqlite has a special syntax, as it does not need a database name or user authentication:

	
 sqlite://file.db
 sqlite://../relative/path/to/file.db
 sqlite://unix(/absolute/path/to/file.db)
 sqlite://windows(c%2A/absolute/path/to/file.db)
